# College Event Planner - Local Storage Version

An AI-powered event management system for colleges with comprehensive features including role-based access, event creation and management, AI-based schedule and task generation, task assignment and progress tracking, and participant registration with attendance tracking.

## Features

- **Local Storage Database**: All data is stored in your browser's localStorage - no external database needed
- **Role-Based Access**: Admin, Faculty, Student, and Volunteer roles with different permissions
- **Event Management**: Create, view, and manage college events
- **AI Task Generation**: Automatically generate event planning timelines with intelligent task scheduling
- **Task Tracking**: Monitor progress with visual dashboards and status updates
- **Registration System**: Students can register for events, track attendance
- **Budget & Venue Management**: Manage event budgets and venue details
- **Fully Responsive**: Works perfectly on mobile, tablet, and desktop devices
- **3D Visual Effects**: Immersive animations and modern design

## Demo Accounts

The application comes with three pre-configured demo accounts:

- **Admin**: admin@college.edu / admin123
- **Faculty**: faculty@college.edu / faculty123
- **Student**: student@college.edu / student123

## Data Storage

All data is stored locally in your browser using localStorage:
- **Users**: Login credentials and user profiles
- **Events**: Event details, dates, venues, budgets
- **Tasks**: AI-generated task timelines with status tracking
- **Registrations**: Event participant registrations

## Features by Role

### Admin & Faculty
- Create new events
- Generate AI-powered event plans
- Update task statuses
- View all registrations
- Manage event budgets and venues

### Students & Volunteers
- Browse all events
- Register for events
- View event details and schedules
- Track event participation

## Technology Stack

- **Frontend**: React with TypeScript
- **Styling**: Tailwind CSS v4
- **Animations**: Motion/React (formerly Framer Motion)
- **UI Components**: Custom component library with glass morphism effects
- **Icons**: Lucide React
- **Storage**: Browser localStorage
- **State Management**: React hooks

## Getting Started

1. Open the application in your browser
2. Use one of the demo accounts to login, or create a new account
3. Explore events, create new ones (if you're Admin/Faculty), and register for events
4. Generate AI task plans for your events
5. Track progress and manage your events

## Note

Since this application uses localStorage, all data is stored locally in your browser. This means:
- Data persists between sessions on the same browser
- Data is specific to the browser and device you're using
- Clearing browser data will reset the application
- Data is not shared between different browsers or devices
